# 🧼 B&W Cleaning Service – Full-Stack Web Application

A professional cleaning service website built as a real-world full-stack project.  
Includes a dynamic quote request system, email integration, and responsive design.

---

## 🌐 Live Demo
👉 https://bwcleaning.balancedben.com

## 🚀 Features

- 📩 “Get a Quote” form with real-time submission
- 📧 Email delivery using Resend API
- 🔄 Redirect flow with success/error feedback
- 🧠 Backend validation and request handling
- 📱 Fully responsive design (mobile-friendly)

## ⚙️ How It Works

1. User fills out the **Get a Quote** form  
2. Form sends a POST request to the server  
3. Backend validates the data  
4. Email is sent using Resend API  
5. User is redirected with success/error feedback
